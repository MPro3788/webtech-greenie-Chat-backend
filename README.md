# webtech-greenie-Chat
Eine Alternative Chat tool zu WhatsApp für HTW-Angehörige im
Datenschutz-komformen und geschützten Bereich, wo Lehrende, Studierenden
und HTW-Mitglieder anhang der Legitimation beim HTW API untereinander kommunizieren können.
Sämtliche Grundokomponenten, die auch im WhatsApp vorhanden sind 
(z.B. P2P-Chat, Gruppenchat, Uploads von Bilder und Dateien), beinhaltet auch dieses Chat Tool
mit dem Unterschied, dass die Nutzung durch Dritter nicht zugelassen ist.

HTW-Mitglieder können nach der Registrierung ausschließlich durch den HTW-Account sich zwischen der
regulären DSGVO oder distrikten DSGVO entscheiden, wobei bei der regulären DSGVO Optionen bestehen wo
der Nutzer/in sich entscheiden kann, welche personenbezogen Daten weiterverarbeitet werden kann.

Außerdem beinhaltet der Chat Tool auch einen Editor, 
mit dem man hochgeladene Dateien und Quellcodes bearbeitet werden kann.
